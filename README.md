# 🎮 Rock Paper Scissors

A classic Rock Paper Scissors game built using **HTML, CSS, and Vanilla JavaScript**.

## ✨ Features

- 🎯 Interactive gameplay
- 🖱️ Click-based user interaction
- 📊 Live score tracking
- 💬 Dynamic game messages
- 🎨 Hover animations and UI effects
- ⚡ DOM Manipulation
- 🎲 Randomized computer moves
- 📱 Clean and responsive interface

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

## 📚 What I Learned

- DOM Manipulation
- Event Listeners
- JavaScript Functions
- Conditional Logic
- Random Number Generation
- Updating the UI dynamically
- Organizing JavaScript code into reusable functions
